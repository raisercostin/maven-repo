# maven-repo
Add the folowing maven repository:

```
<repository>
  <id>raisercostin-github</id>
  <url>https://github.com/raisercostin/maven-repo/tree/master</url>
  <snapshots><enabled>false</enabled></snapshots>
</repository>
```
